# Turf Star - Simulador de Corrida de Cavalos

Este é um jogo de aposta e corrida de cavalos desenvolvido com HTML5, CSS3 e JavaScript puro.

## Como Jogar
1. **Escolha seu Cavalo**: Analise as estatísticas de Velocidade, Aceleração e Resistência.
2. **Faça sua Aposta**: Selecione o valor que deseja apostar no seu favorito.
3. **Torça**: Assista à corrida em tempo real. O sistema utiliza um RNG (Gerador de Números Aleatórios) controlado para garantir emoção.
4. **Gerencie seu Saldo**: Ganhe moedas virtuais e tente bater o seu recorde.

## Estrutura do Projeto
- `index.html`: Estrutura principal do jogo.
- `style.css`: Design premium com Glassmorphism e responsividade.
- `script.js`: Motor do jogo, lógica dos cavalos e sistema de apostas.
- `/assets`: Contém os sprites dos cavalos e o fundo da pista.

## Tecnologias Utilizadas
- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)
- Google Fonts (Outfit)
